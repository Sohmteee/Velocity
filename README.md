# Welcome to Velocity

**Velocity** is the ultimate tool for managing your coding projects effortlessly. Designed for simplicity and power, it brings the complex world of Git version control into a beautiful, easy-to-use desktop application.

Whether you are a seasoned developer or just starting out, Velocity helps you keep track of your work, save your progress, and collaborate with others without the headache of command-line tools.

## üìñ User Guide

### 1. Getting Started

**Installation**

1. Download the latest Windows installer ([Velocity_Setup](https://github.com/Sohmteee/Velocity/releases/tag/v1.0.0)).
2. Run the installer and follow the on-screen instructions.
3. Launch **Velocity** from your desktop or start menu.

**First Run**

* Upon opening Velocity, you will be greeted by the Welcome Screen.
* **Settings**: Click the Gear icon to open Settings. Here you can enter your **Gemini API Key** to enable AI features.

### 2. Managing Projects

**Adding a Project**

* Click the **"New Project"** button on the dashboard.
* **Clone**: Paste a URL from GitHub to download an existing project.
* **Create**: Choose a folder on your computer to start a brand new project.
* **Scan**: Velocity can scan your computer to find existing projects automatically.

**Opening a Project**

* Simply click on any project card in your dashboard to open the **Project Overview**.

### 3. Saving Your Work

**The "Commit" Process**

1. Make changes to your files using your favorite code editor.
2. Velocity automatically detects these changes and lists them in the **"Changes"** tab.
3. **Magic AI Button**: Click the ‚ú® (Sparkle) button. Velocity will read your changes and fill in the message box with a description like "Update login screen styling".
4. Click **Commit** to save this snapshot of your work.

### 4. Syncing with the Cloud

* **Push**: When you are ready to send your saved work to GitHub, click the **Push** button.
* **Pull**: If you are working with a team, click **Pull** to download their latest changes to your computer.
